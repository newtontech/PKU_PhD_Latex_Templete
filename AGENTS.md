# 北京大学博士学位论文LaTeX模板使用指南

## 一、 论文结构要求

### 1.1 文件组织结构

北京大学博士学位论文采用模块化的文件组织结构，便于管理与维护。具体结构如下：

```
PKU_PhD_Latex_Templete/
├── COEmain.tex              # 主控文件
├── AGENTS.md                # 本使用指南
├── front/                   # 前言部分
│   ├── cover.tex           # 封面页
│   ├── approval.tex        # 答辩委员会页
│   ├── original.tex        # 原创性声明
│   ├── abstract.tex        # 中英文摘要
│   └── toc.tex             # 目录页
├── body/                    # 主体部分
│   ├── introduction.tex    # 绪论
│   ├── chapter2.tex        # 第二章
│   ├── chapter3.tex        # 第三章
│   ├── chapter4.tex        # 第四章
│   └── conclusion.tex      # 结论
└── back/                    # 后记部分
    ├── bibliography.tex    # 参考文献
    ├── achievements.tex    # 攻博期间学术成果
    ├── acknowledgement.tex # 致谢
    └── appendix.tex        # 附录
```

## 二、 格式规范

### 2.1 版面设置

#### 2.1.1 基本版式要求
1. **纸张规格**：采用标准A4纸张（210mm×297mm）
2. **版心设置**：
   - 上边界：25mm
   - 下边界：25mm
   - 左边界：30mm（预留装订空间）
   - 右边界：20mm
3. **页眉页脚规范**：
   - 页眉高度：1.5cm
   - 页脚高度：1.75cm

#### 2.1.2 字体与字号规范
- **正文文字**：小四号字体，西文采用Times New Roman，中文采用宋体
- **标题层级**：
  - 一级标题：三号，黑体
  - 二级标题：小三号，黑体
  - 三级标题：四号，黑体
- **图表说明**：五号，宋体

#### 2.1.3 行距与段落设置
- **行距标准**：1.5倍行距
- **段落格式**：首行缩进2个中文字符
- **段落间距**：0（段间无额外间距）

## 三、 编译说明

### 3.1 LaTeX环境配置

#### 3.1.1 必要宏包清单
```latex
\usepackage[UTF8]{inputenc}     % UTF-8编码支持
\usepackage{ctex}               % 中文支持包
\usepackage{amsmath}            % 数学公式环境
\usepackage{amssymb}            % 数学符号扩展
\usepackage{graphicx}           % 图形插入功能
\usepackage{booktabs}           % 专业表格格式
\usepackage{caption}            % 图表标题设置
\usepackage{fancyhdr}           % 页眉页脚设计
\usepackage{geometry}           % 页面几何设置
\usepackage{hyperref}           % 超链接支持
```

#### 3.1.2 编译流程与要求
1. **编译引擎选择**：必须使用XeLaTeX或LuaLaTeX编译引擎以确保中文字体正确显示
2. **LaTeX发行版**：推荐安装完整的TeX Live或MiKTeX发行版
3. **标准编译顺序**：
   - 第一步：`xelatex COEmain.tex`
   - 第二步：`bibtex COEmain`
   - 第三步：`xelatex COEmain.tex`
   - 第四步：`xelatex COEmain.tex`（再次编译确保引用正确更新）

## 四、 论文各部分撰写要求

### 4.1 封面页（cover.tex）
封面页应包含以下关键信息：
- **论文题目**：准确、精炼地反映研究核心内容
- **培养单位**：工学院
- **学科专业**：根据学生实际情况填写
- **指导教师**：注明教授职称及研究方向
- **完成时间**：论文最终完成的年月

### 4.2 摘要（abstract.tex）
#### 4.2.1 中文摘要
- **字数要求**：800-1000汉字
- **内容结构**：
  - 研究背景与学术意义
  - 主要研究内容与目标
  - 研究方法与技术路线
  - 核心创新点与研究结论
- **关键词要求**：3-5个，能够准确概括论文主题

#### 4.2.2 英文摘要
英文摘要应与中文摘要在内容上保持一致，遵循国际学术规范，确保语言表达准确、专业。

### 4.3 正文主体（body/）
#### 4.3.1 绪论（introduction.tex）
绪论部分应系统阐述：
- 研究背景与动机
- 国内外研究现状分析
- 现有研究的局限性分析
- 本研究的学术价值与实际意义
- 研究目标与主要研究内容

#### 4.3.2 主体章节（chapter2-4.tex）
主体章节应包含：
- 理论基础与文献综述
- 研究方法与实验设计
- 研究结果与数据分析
- 深入讨论与结果阐释

#### 4.3.3 结论（conclusion.tex）
结论部分应总结：
- 主要研究成果与创新点
- 研究的不足之处
- 未来研究方向与展望

### 4.4 参考文献（bibliography.tex）
- **文献数量**：建议50-100篇高质量学术文献
- **引用规范**：严格遵循GB/T 7714-2015国家标准
- **文献时效性**：优先引用近5年内发表的文献
- **文献质量**：兼顾经典文献与最新研究成果

## 六、 质量控制与检查清单

### 6.1 自查验证清单
在论文提交前，请逐一核对以下项目：
- [ ] 图表编号连续性检查，确保引用准确无误
- [ ] 参考文献格式一致性验证
- [ ] 页眉页脚设置正确性检查
- [ ] 目录与实际页码对应性核对
- [ ] 文字内容校对，确保无错别字和语法错误
- [ ] 公式编号系统正确性检查
- [ ] 致谢内容得体性审核

### 6.2 常见技术问题解决方案
1. **中文字体显示异常**
   - 解决方案：确保使用XeLaTeX编译引擎
   - 预防措施：检查系统是否安装必要的中文字体

2. **图表位置偏移问题**
   - 解决方案：使用`[!htbp]`参数优化图表位置
   - 最佳实践：避免过多的位置限制参数

3. **参考文献引用错误**
   - 解决方案：检查BibTeX编译过程
   - 调试方法：查看.bbl文件内容确认参考文献处理

4. **目录信息更新滞后**
   - 解决方案：执行完整的两次编译流程
   - 注意事项：确保所有引用都正确编译

## 七、 论文提交准备

### 7.1 最终提交材料清单
在论文答辩通过后，需准备以下提交材料：
1. **正式论文**：PDF格式的完整论文文档

## 八、 编译环境配置与使用须知

### 8.1 重要操作提示
为确保模板正常使用，请严格遵守以下规定：
1. **文件完整性**：请勿随意修改.tex模板文件及其引用的配置文档
2. **依赖包管理**：如遇到缺失的宏包或字体，请及时安装相应组件
3. **系统权限使用**：审慎使用sudo权限，优先考虑用户级安装方案
4. **编译工作流**：严格遵循"XeLaTeX → BibTeX → XeLaTeX → XeLaTeX"的编译顺序
5. **环境调试**：持续尝试不同解决方案，直至编译环境完全配置成功
6. **版本控制策略**：配置.gitignore文件，确保中间文件（如.aux、.toc、.log等）不被Git追踪，但需保留.tex和.pdf文件

### 8.2 使用注意事项
- **标准依据**：本模板严格遵循北京大学研究生院最新学位论文格式要求
- **动态更新**：具体格式要求以研究生院官方发布的最新通知为准
- **数据安全**：建议建立定期的源文件备份机制
- **质量审核**：论文正式提交前，必须通过学院的预审环节
